Log In Sign Up

TRIGGERS

Log In button

  • Hover On: The button's background color will become slightly darker.
  • Hover Off: The button goes back to its original color.

RULES

Log In button hover on:

When the Log In button is hovered over, the button's background color and the text color within the button will darken slightly.


Log In button hover off:

When the Log In button is hovered off, the button's background color and the text color within the button will revert to original settings.


Log In button click:

When the Log In button is clicked, the user will be redirected to a log in screen.

LOOPS AND MODES

The hover on, hover off rules repeat whenever the user hovers on or off of the Log In button.

There are no modes.
