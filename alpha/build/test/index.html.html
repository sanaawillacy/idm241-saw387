Log In Sign Up

TRIGGERS

When a user hovers or clicks on the “Log in” or “Sign up” buttons, the microinteraction begins.

RULES

1. Hover changes button color.
2. Clicking navigates to the corresponding page.

FEEDBACK

The user sees a color transition when hovering, giving visual confirmation that the button is active.

LOOPS & MODES

This interaction repeats each time the page is loaded or hovered again. No additional modes apply.
