Visit Original Robinhood Page

This GIF shows how the existing Log In button microinteraction looks on the Robinhood homepage.Please note that it's hard to see because the button's color is dark already.

TRIGGERS

Log In button

  • Hover On: The button's background color will become slightly darker
  • Hover Off: The button goes back to its original color

RULES

LogIn button: When the Log In button is clicked, the button's background color and the text color within the button will darken slightly.

Loops and Modes

The microinteraction repeats whenever the user interacts with the Log In button. There are no modes.
