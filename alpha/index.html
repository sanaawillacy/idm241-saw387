Alpha Microinteraction: Robinhood Log In Button

Robinhood Start Page

The microinteraction that I chose is the Log In button on the Robinhood startup page

Visit Original Robinhood Page

This GIF shows how the existing Log In button microinteraction looks on the Robinhood homepage.Please note that it's hard to see because the button's color is dark already.

Triggers

The microinteraction is triggered when the user hovers or clicks on the Log In button in the top right corner. The button becomes slightly darker when hovering over it but it's hard to see.

Rules

When the Log In button is clicked the button's background color and the text color within the button will darken slightly.

Loops and Modes

The microinteraction repeats whenever the user interacts with the Log In button. There are no modes.
